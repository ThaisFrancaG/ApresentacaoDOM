Quando uma página é carregada, o navegador cria um DOM dessa página. O DOM, por sua vez, define a estrutura lógica através da qual documentos podem ser acessados e manipulados!

Para realizar a manipulação do DOM, eu posso usar outras linguagens de programação junto ao HTML/XML. Uma das mais comuns utilizadas em páginas da web é o javascript.

Mas, é preciso ter noção da diferença entre DOM e Script. O DOM não existe através do script, mas é o script quem permite interação e modelagem do DOM, de forma a o tornar dinâmico

Mas afinal, se o documento é o HTML é o documento, que contem os objetos, e o navegador permite que ele seja visualizado como um modelo que segue certo conjunto de lógicas, qual o nome do local onde o documento é exibido?

Elementos são apenas um dos tipos de Nodes ou Nós que compõem um documento.

Resumindo:

  • "Node" é qualquer tipo de objeto dentro da DOM. Toda webPage é feita por vários Nodes
  • "Element" é um tipo específico de Node
